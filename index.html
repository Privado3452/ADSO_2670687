Concepto de GIT

Git es sistema de control de versiones, en el cual podemos tener una copia de todo los cambios que le hagamos a nuestro codigo,tambien podremos conocer quien es la persona que esta realizando los cambios en dicho codigo

los principales comandos de git

  • GIT INIT: este comando funciona para iniciar un proyecto con git.
  • GIT ADD .: me permite agregar cambios al area de reparacion.
  • GIT CONFIG --GLOBAL USER.MANE: Es para colocar el nombre de la persona que esta realizando los cambios al codigo
  • GIT CONFIG --GLOBAL USER.EMAIL: Es para colocar el correo de la persona que esta realizando los cambios al codigo
  • GIT STATUS: este comando es para conocer el estado del actual del archivo.
  • GIT COMMIT -M "": este comando funciona para agregar un comentario de lo que se realizando en el repositorio.
  • GIT BRANCH: este comando funciona para visualizar en que rama nos encontramos.
  • GIT CHECKOUT: este comando funciona para desplazarse entre las ramas creadas por git branch.
  • GIT PUSH -U ORIGIN MAIN: este comando funciona para activar el repositorio o el branch.
